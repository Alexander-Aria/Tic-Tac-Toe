# Tic Tac Toe

A simple terminal-based tic-tac-toe game written in C++.

## Features

- Two-player gameplay
- Score saving
- Terminal interface

## Requirements

- Linux
- g++
- Make

## Build

```bash
make
```

## Run

```bash
make run
```

## Notes

For manual build and run, instructions are in `Makefile`. 
Scores are stored locally in `data/Score/`.
